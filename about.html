Mi az apaPong?

Az apaPong egy klasszikus kétjátékos Pong játék, amelyet a modern webes technológiákkal fejlesztettünk. A játék célja, hogy két játékos egymás ellen játszhasson egy egyszerű, de izgalmas ping-pong stílusú játékban.

A játék teljes mértékben a böngészőben fut, nincs szükség telepítésre vagy regisztrációra. Egyszerűen nyisd meg az oldalt és kezdj el játszani!

Játékverzió

Verzió: 1.0.0

Fejlesztés dátuma: 2024

Licenc: Nyílt forráskódú, személyes használatra

Főbb jellemzők

  • Kétjátékos mód: Helyi többjátékos játék egyetlen eszközön
  • Testreszabható játékosnevek: Adj nevet mindkét játékosnak
  • Színválasztás: Válassz színt az ütőknek és a labdának
  • Dinamikus labdasebesség: A labda gyorsul minden ütés után
  • Hanghatások: Hangok a labda ütközésekor
  • Pontszám rendszer: Az első 5 pontot elérő játékos nyer
  • Visszaszámlálás: Minden új kör előtt visszaszámlálás

Technológiák

A játék a következő technológiákkal készült:

Kaplay.js v3001.0.12 JavaScript (ES6+) HTML5 CSS3 Vitest

A Kaplay.js egy modern, könnyű JavaScript játékfejlesztő keretrendszer, amely lehetővé teszi a gyors és hatékony játékfejlesztést.

Vezérlés

  • Bal játékos: W (fel) / S (le)
  • Jobb játékos: ↑ (fel) / ↓ (le)
  • Játék közben: ESC (menü) / R (újraindítás)

Projekt struktúra

A projekt moduláris felépítésű:

  • core/ - Alapvető játéklogika
  • scenes/ - Játékjelenetek
  • css/ - Stíluslapok
  • tests/ - Unit tesztek

Fejlesztés és tesztelés

A projekt tartalmaz unit teszteket a Vitest keretrendszerrel. A tesztek futtatásához:

npm install
npm test
npm run test:coverage

Kapcsolat

Kérdéseid, javaslataid vagy hibajelentéseid esetén kérjük, használd a GitHub repository Issues szekcióját.

Köszönet
